# 01

Java აპლიკაციების Command Line-ის გამოყენებით კომპილაცია და გაშვება.

- კომპილაცია და გაშვება

```shell
java Main.java
```

- მარტო კომპილაცია

```shell
javac Main.java
```

- დაკომპილირებულის გაშვება

```shell
java Main
```
